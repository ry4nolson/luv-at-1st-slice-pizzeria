# Luv At 1st Slice Pizzeria

Next.js website for Luv At 1st Slice Pizzeria — NY style pizza slices, whole pies, icees, and homemade Jamaican beef patties. Beaumont, TX.

## Getting Started

```bash
npm install
npm run dev
```

Deploy easily on Netlify or Vercel — this is a standard Next.js 15 App Router project.
